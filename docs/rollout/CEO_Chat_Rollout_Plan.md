# CEO Chat Rollout Plan

Rollout rings:

1. Local deterministic CEO chat with durable threads.
2. Internal read-only CEO beta.
3. Context-aware strategy and simulation beta.
4. Supervised draft beta.
5. Governed paper automation pilot.

Exit criteria:

- Focused CEO chat tests pass.
- UI widget remains globally mounted and accessible.
- No live execution path is reachable from chat.
- Audit metadata is present on persisted assistant messages.
- Operators can roll back by disabling the `/api/ai-chat/*` route or hiding the launcher.
